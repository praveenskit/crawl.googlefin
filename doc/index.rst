구글 파이낸스 크롤러
==============================================

개요
---------
google finance 의 기능인 stock screener 와 get prices 를 사용해 주식 가격 데이터를 크롤링 합니다.
크롤링 된 데이터는 AWS RDS 에 업로드 합니다.

테이블
--------
* symbol_exchange

|  column  |    type    |
+----------+------------+
|  company |   varchar  |
+----------+------------+
|  symbol  |   varchar  |
+----------+------------+

+----------+------------+
